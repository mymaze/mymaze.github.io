### 玩法说明

- 基础操作

| 按键          | 效果    |
|-------------|-------|
| Esc / Tab   | 弹出对话框 |
| WASD / ↑↓←→ | 玩家移动  |
| - / _       | 提示    |
| + / =       | 自动寻路  |

- 随机传送

迷宫会生成传送门，玩家进入传送门将会随机传送迷宫的任意位置

- 变换莫测

迷宫会每隔一段时间重新生成

- 黑夜降临

玩家的可见度将会变得狭窄

- 行尸走肉

迷宫会投放一定数量的丧尸，丧尸发现玩家会快速扑向玩家

- 地图扩展

迷宫尺寸默认为屏幕大小，地图扩展后宽高将会是原来的 N 倍
